Fonctions de validation de formulaires, côté serveur.

Auteur
Thibault ROSALIE, Théo BIREMBAUX

Ce fichier contient les fonctions de validation des données au niveau du serveur. Ces fonctions permettent de vérifier la cohérence des données sur les formulaires présents sur l'application.
